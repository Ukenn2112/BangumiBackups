# BangumiBackups 个人数据备份

- 将 [Bangumi.tv](bgm.tv) 上的个人数据进行备份

# 功能

- [x] 收藏记录备份

# 使用方法

- 准备 Access Token

  前往 Bangumi 官方 [Access Token 管理平台](https://next.bgm.tv/demo/access-token) 生成

- 安装依赖
  > **Warning** 需要 Python >= 3.9
  ```
  pip3 install -r requirements.txt
  ```

- 运行
  ```
  python3 main.py
  ```
